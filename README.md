# 🎮 俄罗斯方块 Tetris

一个用纯 HTML / CSS / JavaScript 实现的俄罗斯方块小游戏，**零依赖、单文件**，双击即可在浏览器中运行。

![Tetris](https://img.shields.io/badge/游戏-俄罗斯方块-7dd3fc) ![HTML5](https://img.shields.io/badge/HTML5-✓-e34f26) ![No Dependencies](https://img.shields.io/badge/依赖-零-4ade80)

---

## ✨ 功能特性

- 🎯 **7 种标准方块**（I / O / T / S / Z / J / L），各有独立配色
- 🔄 **墙踢旋转**：靠墙旋转时会自动尝试横向偏移，操作更顺手
- 👻 **幽灵投影**：当前方块的落点以半透明轮廓预显示
- 👀 **下一个方块预览**：右侧面板提前展示下一个方块
- 📊 **实时计分**：分数 / 消除行数 / 等级同步显示
- ⚡ **难度递增**：每消除 10 行升一级，下落速度加快
- ⏸️ **暂停 / 继续**：随时暂停，继续游戏
- 💯 **多种下落模式**：逐格下落、软降（加速）、硬降（一落到底）

---

## 🚀 快速开始

无需安装任何环境，两种方式任选其一：

### 方式一：直接双击打开
下载 `tetris.html` → 双击文件 → 用默认浏览器打开即可游玩。

### 方式二：克隆仓库
```bash
git clone https://github.com/monks1980/monkey.git
```
然后双击 `monkey/tetris.html` 打开。

> 💡 推荐使用 Chrome / Edge / Firefox 等现代浏览器。

## 🕹️ 操作指南

| 按键 | 功能 |
|:----:|------|
| **← →** | 左右移动 |
| **↑** | 旋转方块 |
| **↓** | 软降（加速下落，每格 +1 分） |
| **空格** | 硬降（直落到底，每格 +2 分） |
| **P** | 暂停 / 继续 |

---

## 📈 计分规则

| 消除行数 | 得分（× 当前等级） |
|:--------:|:------------------:|
| 1 行（单消） | 100 |
| 2 行（双消） | 300 |
| 3 行（三消） | 500 |
| 4 行（四消 / Tetris） | 800 |

- **升级条件**：每累计消除 **10 行** 升一级
- **速度变化**：等级越高，方块下落越快（最快 80ms / 格）
- **硬降奖励**：硬降每格额外加 2 分

---

## 🛠️ 技术实现

- **渲染**：原生 Canvas 2D API，无任何图形库
- **逻辑**：经典矩阵碰撞检测 + 简易墙踢（wall kick）算法
- **界面**：纯 CSS，渐变背景 + 毛玻璃面板，响应式布局
- **存储**：当前版本无本地存储，刷新即重置（计划中）

整个游戏封装在单个 `tetris.html` 文件中（约 400 行），结构清晰，适合作为 Canvas 游戏开发的入门参考。
